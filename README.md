## 1-프로젝트 소개

### 1-1 개요
`Realm과 API 활용 프로젝트`
- Realm과 API를 활용하여 데이터 베이스 구조 파악 및 활용

### 1-2 주요목표
`과제 전형 미리 경험해보기`
- 데이터 스키마 설계
- 재사용성을 고려한 UI
- 기능과 역할에 따른 코드 구조

### 1-3 개발환경
- 활용기술 외 키워드
  - iOS : iOS 13.0
  - Network: 네이버 API 
  - UI : CodeUI
  - Layout : SnapKit
- 라이브러리
  - SnapKit
  - Alamofire
  - Kingfisher
  - Realm
  - RealmDatabase
